## Restaurant UI/UX Web Application

A responsive restaurant website built using React.js with a focus on clean UI/UX design and user-friendly navigation.

## Features
- Modern and responsive UI design
- Menu display section
- Reusable React components
- Mobile-friendly layout

## Tech Stack
- React.js
- HTML5
- CSS3
- JavaScript

## What I Learned
- React component structure
- Props and state management
- UI/UX design principles
- Responsive web design
